## 👋 About Me

Hey! I'm a **Full-Stack Developer** with **6+ years** of experience in web and mobile development. Over that time I've taken **8 products from zero to production** — across SaaS, FinTech, E-commerce, Telehealth, and AI — working with cross-functional remote teams across Ukraine and Europe.

My core stack: **React / Next.js / React Native** on the frontend and **Node.js / NestJS / GraphQL** on the backend, deployed on **AWS / Azure / GCP**.

> I take ownership end-to-end — from architecture decisions to production monitoring. I communicate clearly, document what matters, mentor when needed, and consistently deliver without hand-holding.

<img src="https://raw.githubusercontent.com/andreasbm/readme/master/assets/lines/rainbow.gif" width="100%"/>

## 🚀 What I've Delivered in Practice

- ⚡ Reduced page load times by **35%** and API response times by **30%** on production systems serving real users
- 💳 Improved checkout conversion by **~20%** through frontend optimization on a telehealth platform
- 🔄 Built and maintained scalable real-time systems with **WebSockets** and **GraphQL** across SaaS platforms
- 📱 Delivered mobile apps with **offline-first** architecture and push notification systems using **React Native + Expo**
